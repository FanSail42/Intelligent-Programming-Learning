## 二、抽象类（Abstract Class）

### 2.1 特点

- 用 `abstract` 修饰，**不能** `new` 抽象类。  
- 可含抽象方法（无方法体）与具体方法。  
- 子类必须实现全部抽象方法，除非子类也是抽象类。

```java
public abstract class Account {
    protected double balance;

    public abstract void withdraw(double amount);

    public void deposit(double amount) {
        if (amount > 0) balance += amount;
    }
}
```

### 2.2 何时使用

- 多个子类共享**相同状态与代码**（如 balance、deposit）。  
- 需要定义**模板流程**，部分步骤由子类实现（模板方法模式雏形）。

## 三、接口（Interface）

### 3.1 特点

- 接口描述**能力/契约**，字段默认 `public static final`。  
- 类通过 `implements` 实现接口，可多实现。  
- Java 8+ 可有 `default` / `static` 方法。

```java
public interface Flyable {
    void fly();

    default void land() {
        System.out.println("安全着陆");
    }
}

public class Duck extends Animal implements Flyable {
    @Override
    public void fly() {
        System.out.println("鸭子飞行");
    }
}
```

### 3.2 抽象类 vs 接口

| 对比项 | 抽象类 | 接口 |
|--------|--------|------|
| 继承/实现 | extends（单继承） | implements（多实现） |
| 成员变量 | 可有实例字段 | 仅 public static final |
| 构造器 | 有 | 无 |
| 典型用途 | 模板 + 共享代码 | 能力约定、解耦 |

## 四、Comparable 与排序

```java
public class Student implements Comparable<Student> {
    private String id;
    private int score;

    @Override
    public int compareTo(Student other) {
        return Integer.compare(this.score, other.score); // 按分数升序
    }
}

// 使用
List<Student> list = ...;
Collections.sort(list);
// 或 Java 8+
list.sort(Comparator.comparingInt(Student::getScore).reversed());
```

## 五、综合项目：简易图书借阅系统

### 5.1 需求

1. **Book**：ISBN、书名、作者、是否借出。  
2. **Reader**：读者 ID、姓名、已借数量（最多 3 本）。  
3. **Borrowable** 接口：`boolean borrow(Reader r)`、`void returnBook(Reader r)`。  
4. **Library** 实现借还逻辑，维护书籍与读者列表。  
5. 控制台菜单：查询、借阅、归还、列出可借图书。

### 5.2 类图（文本）

```
        <<interface>>
         Borrowable
              ^
              |
          Library
         /    |    \
     Book  Reader  (List)
```

### 5.3 核心代码骨架

```java
public interface Borrowable {
    boolean borrow(Reader reader);
    void returnBook(Reader reader);
}

public class Book {
    private final String isbn;
    private final String title;
    private boolean borrowed;

    public boolean isAvailable() { return !borrowed; }
    void markBorrowed() { borrowed = true; }
    void markReturned() { borrowed = false; }
}

public class Reader {
    private final String id;
    private final String name;
    private int borrowedCount;
    private static final int MAX = 3;

    public boolean canBorrow() { return borrowedCount < MAX; }
    public void onBorrow() { borrowedCount++; }
    public void onReturn() { borrowedCount--; }
}

public class Library implements Borrowable {
    private final List<Book> books = new ArrayList<>();
    private final Map<String, Reader> readers = new HashMap<>();

    @Override
    public boolean borrow(Reader reader) {
        if (!reader.canBorrow()) return false;
        for (Book b : books) {
            if (b.isAvailable()) {
                b.markBorrowed();
                reader.onBorrow();
                return true;
            }
        }
        return false;
    }

    @Override
    public void returnBook(Reader reader) {
        // 简化：归还第一本已借图书
        for (Book b : books) {
            if (!b.isAvailable()) {
                b.markReturned();
                reader.onReturn();
                return;
            }
        }
    }
}
```
