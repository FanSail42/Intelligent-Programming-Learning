## 二、继承（Inheritance）

### 2.1 语法

```java
public class Dog extends Animal {
    public Dog(String name) {
        super(name);  // 调用父类构造器，必须在子类构造器第一行
    }

    @Override
    public void speak() {
        System.out.println(name + "：汪汪汪");
    }
}
```

### 2.2 继承规则

- 子类继承父类**非 private** 的成员。  
- 子类**不继承**父类构造器，但可通过 `super(...)` 调用。  
- Java **单继承**：一个类只能 extends 一个直接父类。  
- `Object` 是所有类的根类：`toString()`、`equals()`、`hashCode()` 等。

### 2.3 super 关键字

| 用法 | 含义 |
|------|------|
| `super()` | 调用父类构造器 |
| `super.method()` | 调用父类被重写的方法 |
| `super.field` | 访问父类被隐藏的同名字段（少用） |

## 三、多态（Polymorphism）

### 3.1 编译时多态：方法重载（Overload）

同一类中，方法名相同、参数列表不同（个数/类型/顺序）。与继承无必然关系。

```java
void print(int x) { }
void print(String s) { }
```

### 3.2 运行时多态：向上转型 + 方法重写

```java
Animal a = new Dog("旺财");  // 向上转型
a.speak();                   // 运行期绑定 → Dog.speak()
```

**三要素**：

1. 继承（或实现接口）  
2. 子类重写父类方法  
3. 父类引用指向子类对象  

### 3.3 向下转型与 instanceof

```java
if (a instanceof Dog d) {  // Java 16+ 模式匹配
    d.fetch();             // Dog 特有方法
}
```

错误转型会抛出 `ClassCastException`。

## 四、完整示例：图形面积

```java
public abstract class Shape {
    public abstract double area();
}

public class Circle extends Shape {
    private final double radius;
    public Circle(double radius) { this.radius = radius; }

    @Override
    public double area() {
        return Math.PI * radius * radius;
    }
}

public class Rectangle extends Shape {
    private final double w, h;
    public Rectangle(double w, double h) { this.w = w; this.h = h; }

    @Override
    public double area() {
        return w * h;
    }
}

// 多态调用
public class Main {
    public static void printArea(Shape s) {
        System.out.println(s.area());
    }

    public static void main(String[] args) {
        Shape[] shapes = { new Circle(2), new Rectangle(3, 4) };
        for (Shape s : shapes) {
            printArea(s);
        }
    }
}
```

## 六、设计原则提示

- **里氏替换**：子类对象应能替换父类对象使用，不改变程序正确性。  
- **不要过度继承**：优先组合（has-a）而非继承（is-a），当“只是一个部件”时。  
- 重写 `equals/hashCode` 时保持契约一致（尤其使用 HashMap 时）。
