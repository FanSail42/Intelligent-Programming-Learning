# M07_教师教学支持

> **Phase**：4 | **优先级**：P1 | **更新**：2026-06-11（**MVP 已编码验收**）

## 1. 业务目标

为教师提供 **班级视角** 的学情统计，与学生端 M06 仪表盘互补：展示选课人数、班级错题分布、知识点薄弱情况与近 7 日学习事件趋势，支撑答辩演示「教师教学支持」能力。

完整版（作业布置、AI 审核队列、旁听会话）归属 Phase 4+ 扩展，本 MVP 聚焦 **单课程学情概览**。

## 2. 用例与角色

| 用例 | 角色 | 说明 |
|------|------|------|
| 查看课程学情概览 | teacher / admin | 班级汇总指标 + ECharts |
| 访问他人课程学情 | teacher | 禁止（403） |
| 作业 CRUD | teacher | Phase 4+ 延后 |
| AI 回答审核队列 | teacher | Phase 4+ 延后 |

## 3. 业务规则

### 3.1 数据范围

- 仅统计 **已选课** 学生（`course_student.deleted=0`）的学习事件与错题本
- 错题本 `course_id` 过滤逻辑与 M06 `apply_wrong_book_course_filter` 一致（含 KP 回填）
- 近 7 日趋势：按 **本地日期** 对齐今日，与学生仪表盘一致

### 3.2 班级薄弱知识点

1. 对选课学生批量刷新 `user_kp_mastery`（复用 `mastery.py`）
2. 取该课 `knowledge_point` 下各 KP 的 **平均掌握度**
3. 按平均分升序取 Top8，供 `WeakKpChart` 展示

### 3.3 权限

- `teacher`：仅 `course.teacher_id == 当前用户`
- `admin`：可查看任意已发布/草稿课程
- `student`：403
