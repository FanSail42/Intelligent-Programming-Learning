# M09_账户管理与 AI 配置

> **Phase**：4 | **优先级**：P1 | **更新**：2026-06-11（**AI 模型管理子模块**）

## 1. 业务目标

为 **管理员** 提供系统管理能力：学生/教师账号维护、管理概览、操作日志，以及 **AI 模型与密钥配置**（对接阿里云百炼）。

## 3. AI 模型管理规则

- 模型列表来自预置百炼常用项 + **管理员自定义模型**（`ai.custom_chat_models` JSON）
- 自定义模型 ID 需与百炼控制台 Model ID 一致（如 `kimi-k2.6`）
- 配置写入 `sys_config`，API Key 使用 Fernet **加密存储**（密钥由 `CONFIG_ENCRYPTION_KEY` 或 `JWT_SECRET_KEY` 派生）
- 未在 DB 覆盖的项回退至 `.env`（`LLM_API_KEY` 等）
- 保存后立即刷新运行时缓存，AI 对话/代码讲解/Embedding 自动使用新配置
- 每用户日调用上限：`llm_daily_limit`（Redis `rate:llm:{userId}`）
- Token 消耗统计：Redis 累计 + 近 7 日趋势（`stats:llm:*`）

## 4. API 列表

### 账户与日志

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/admin/overview` | 管理概览统计 |
| GET | `/api/v1/admin/logs` | 操作日志分页 |
| GET | `/api/v1/admin/users?role=` | 分页列表（**role 必填**） |
| POST | `/api/v1/admin/users` | 创建账号 |
| PUT | `/api/v1/admin/users/{id}?role=` | 更新状态/密码 |
| DELETE | `/api/v1/admin/users/{id}?role=` | 软删除 |

### AI 配置

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/admin/ai/models` | 百炼模型目录 + 自定义模型 |
| POST | `/api/v1/admin/ai/models/custom` | 添加自定义对话模型 |
| DELETE | `/api/v1/admin/ai/models/custom/{id}` | 删除自定义模型 |
| GET | `/api/v1/admin/ai/config` | 当前配置（密钥脱敏） |
| PUT | `/api/v1/admin/ai/config` | 更新模型/密钥/限额 |
| GET | `/api/v1/admin/ai/usage` | 全局 Token 与调用统计 |
| GET | `/api/v1/admin/ai/usage/students` | **学生 Token 用量分页** |
