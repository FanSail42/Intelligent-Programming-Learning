# 慧编学伴开题报告 — 英文参考文献导读

共 **5 篇**，服务于「**国外研究动态**」撰写。与中文 28 篇互补：中文侧重国内教改与系统实践，英文侧重国际 CS 教育顶会/arXiv 前沿。

## [E1] CodeHelp: Using LLM with Guardrails for Scalable Support in Programming Classes

- **作者**：Mark Liffiton; Brad Sheese; Jaromir Savelka; Paul Denny
- **来源**：Koli Calling 2023（ACM）
- **DOI**：10.1145/3631802.3631830
- **PDF**：`PDF/01_CodeHelp_LLM_Guardrails_Programming_Classes.pdf`

**核心内容**：提出 CodeHelp——带**护栏（guardrails）**的 LLM 编程辅导工具，通过多轮 Prompt 流水线**不直接给出完整代码**，在 52 人首年课程中部署 12 周；学生认可其可用性与错误排查帮助，教师认为易部署且**补充而非替代**人工辅导。

**撰写用途（国外动态2）**：国外 LLM 编程辅导工具的**代表性系统**；与慧编学伴 M04「可控辅导」思路一致。

**可引要点**：guardrails；on-demand assistance without revealing solutions；instructor features。

## [E2] Patterns of Student Help-Seeking When Using an LLM-Powered Programming Assistant

- **作者**：Brad Sheese; Mark Liffiton; Jaromir Savelka; Paul Denny
- **来源**：ACE 2024（ACM）
- **DOI**：10.1145/3636243.3636249
- **PDF**：`PDF/02_Patterns_Student_Help_Seeking_LLM_Programming_Assistant.pdf`

**核心内容**：基于 CodeHelp 在真实课堂中收集 **2500+ 条**学生查询，人工分类求助类型并分析查询特征，揭示学生实际如何使用 LLM 编程助手。

**撰写用途（国外动态2）**：**实证研究**——国外不仅做系统，还分析求助行为；可对比国内 [13] AIOJ 效果评估。

**可引要点**：help-seeking patterns；authentic classroom deployment；over-reliance concerns。

## [E3] Integrating AI Tutors in a Programming Course (RAGMan)

- **作者**：Iris Ma; Alberto Krone Martins; Cristina Videira Lopes
- **来源**：arXiv:2407.15718（UCI；SIGCSE Virtual 2024 同源工作）
- **PDF**：`PDF/03_Integrating_AI_Tutors_Programming_Course_RAGMan.pdf`

**核心内容**：RAGMan 利用 **RAG + 严格指令**构建作业级 AI 助教；在 455 人入门编程课中部署 5 个作业助教，范围内回答准确率 **98%**，78% 使用者认为有助于学习。

**撰写用途（国外动态3）**：与慧编学伴 **M03 RAG + 多课程/多资料** 最接近的国外案例。

**可引要点**：homework-specific AI tutors；RAG alignment；98% accuracy within scope。

## [E4] Teaching with AI: A Systematic Review of Chatbots, GenAI, and ITS in Programming Education

- **作者**：Said Elnaffar; Farzad Rashidi; Abedallah Zaid Abualkishik
- **来源**：International Journal of Learning, Teaching and Educational Research, 2026, 25(1)
- **DOI**：10.26803/ijlter.25.1.1
- **PDF**：`PDF/04_Teaching_with_AI_SLR_Programming_Education.pdf`

**核心内容**：系统综述 **58 篇**（2022–2025）编程教育 AI 研究，归纳为 chatbots、GenAI、ITS 三类；94.83% 研究以增强编程支持为目标；93.10% 提到部署障碍，65.52% 提到过度依赖与浅层学习。

**撰写用途（国外动态1）**：**国外研究全景**——开段首选；可与国内 [E4] 对照写「国际同样关注 over-reliance 与 academic integrity」。

**可引要点**：58 studies；three categories of AI agents；setup barriers；human oversight。

## [E5] Small Language Models for Curriculum-based Guidance

- **作者**：Konstantinos Katharakis; Sippo Rossi; Raghava Rao Mukkamala
- **来源**：arXiv:2510.02347, 2025
- **PDF**：`PDF/05_SLM_Curriculum_based_Guidance.pdf`

**核心内容**：基于 **RAG + 开源 SLM**（LLaMA 3.1、Granite、Gemma 等）构建课程对齐 AI 助教，对标 GPT-4o；强调**隐私、成本、可持续**与引用溯源（citations in responses）。

**撰写用途（国外动态3）**：补充 RAGMan——国外亦探索**课程材料检索 + 小模型本地部署**；对应慧编学伴 Chroma 向量库与 API 密钥治理。

**可引要点**：curriculum-based RAG；SLMs match LLMs with retrieval；privacy-preserving local deployment。
