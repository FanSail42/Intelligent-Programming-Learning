## 摘要

This paper examines how students use an LLM-powered programming assistant (CodeHelp) that provides on-demand help without revealing solutions directly. Deployed for 12 weeks in an introductory course (n=52), the study collected more than 2,500 student queries, manually categorized assistance types, and analyzed query characteristics to understand real-world help-seeking behavior.
