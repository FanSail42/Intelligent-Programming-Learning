## 八、快速命令索引

```bash
# 中间件
docker compose up -d

# 后端
cd backend && alembic upgrade head && uvicorn app.main:app --reload --port 8000

# 种子
python scripts/seed_demo.py
python scripts/seed_dashboard_demo.py

# 测试
cd backend && pytest -q
cd frontend && npm run build

# 前端
cd frontend && npm run dev
```
