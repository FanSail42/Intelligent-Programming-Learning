## 三、分模块功能清单

### M01 认证与系统管理

| 功能 | 说明 | 路由/API |
|------|------|----------|
| 登录 / 刷新 / 登出 | JWT + Redis 黑名单 | `POST /api/v1/auth/login|refresh|logout` |
| 当前用户 | 获取登录态 | `GET /api/v1/auth/me` |
| **个人中心** | 用户名/密码自助修改、账户概览 | `GET/PATCH /api/v1/auth/profile/*` |
| 三角色 RBAC | student / teacher / admin | 路由守卫 + 依赖注入 |
| 健康检查 | MySQL/Redis/Chroma 探活 | `GET /health` |

**前端**：`/login`（专属背景 + 品牌 Logo）、`/403`、`/profile`（三类用户通用）

---

### M02 课程与选课

| 功能 | 说明 |
|------|------|
| 课程 CRUD | 教师创建/编辑/发布 |
| 选课 / 退课 | 学生浏览与我的课程 |
| 发布审核 | 草稿 → 已发布 |
| 演示种子 | 50 门课 + 3 门 dashboard 代表课 |

**前端**：`/student/courses`、`/student/browse`、`/teacher/courses`

---

### M03 课程知识库

| 功能 | 说明 |
|------|------|
| 资料上传 | PDF / TXT / MD / PPTX |
| 解析流水线 | 上传 → 解析 → 向量化 → READY |
| Chroma 检索 | 按课程 ID 向量 isolation |
| 资料仓库 | 格式仓 + 课程仓 + 分派/克隆 |
| 重试 / 删除 | 失败资料可重试，软删除清向量 |

**前端**：`/teacher/materials`、`/teacher/warehouses`、`/teacher/warehouses/:id`

---

### M04 AI 对话辅导

| 功能 | 说明 |
|------|------|
| SSE 流式对话 | 多轮上下文 |
| RAG 检索 | 引用 Top3 + 相关性过滤 |
| 课程导向 | 切换课程新会话、推荐问题 |
| 代码示例 | 按课程语言 C++/Java/Python |
| 空会话清理 | 离开自动删除无消息会话 |

**前端**：`/student/chat`（DeepSeek 风格布局 + Markdown 渲染）

---

### M05 智能代码讲解

| 功能 | 说明 |
|------|------|
| 四语言提交 | C++ / Java / Python / JavaScript |
| LLM 分级分析 | JSON 结构化结果 |
| Monaco 编辑器 | 分栏展示 + 参考代码 |
| 错题本联动 | 严重/失败问题自动入库 |

**前端**：`/student/code`、`/student/code/history`

---

### M06 学习分析与推荐

| 功能 | 说明 |
|------|------|
| 学习事件埋点 | code_submit / error / chat_no_context |
| 错题本 | 列表、筛选、图表、标记掌握 |
| 掌握度公式 | Redis 缓存仪表盘 |
| 7 日趋势 | 本地时区对齐，末位为「今日」 |
| 薄弱 KP 图 | 阶梯对比 ECharts |
| 规则推荐 Top3 | 非 LLM，基于错题/KP |

**前端**：`/student/dashboard`、`/student/wrong-book`
