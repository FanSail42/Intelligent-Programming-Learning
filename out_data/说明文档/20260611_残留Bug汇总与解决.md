## 三、Playbook（按场景排查）

### 3.1 场景：管理概览 / AI 管理 / 个人中心 API 403 或 502

**现象**：浏览器 Network 显示 `403 Forbidden` 或 `502 Bad Gateway`；OpenAPI 无 `/api/v1/admin/overview`。

**根因**：8000 端口被**旧版 uvicorn** 占用，或前端 `VITE_API_BASE` 指向错误实例。

**解决**：

```powershell
# 1. 查占用
netstat -ano | findstr :8000

# 2. 结束旧 PID（替换为实际 PID）
taskkill /PID <pid> /F

# 3. 启动唯一后端实例
cd backend
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# 4. 验证 OpenAPI 含新路由
# 浏览器打开 http://127.0.0.1:8000/openapi.json
# 搜索 admin/overview、admin/ai/models、auth/profile/summary

# 5. 确认 frontend\.env.development
# VITE_API_BASE=http://127.0.0.1:8000

# 6. 修改 .env 后必须重启 npm run dev
```

**临时方案**：后端改 `--port 8002`，同步 `VITE_API_BASE=http://127.0.0.1:8002`。
