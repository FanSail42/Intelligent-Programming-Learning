## 一、功能概述

M06 采集学生在 **代码讲解（M05）**、**AI 对话（M04）** 等行为，形成 **错题本** 与 **知识点掌握度**，通过 **规则引擎**（非 LLM）给出复习建议与资料推荐，支撑学生端 **学习仪表盘** 与 **错题本** 页面。

---

## 二、用户功能

### 2.1 学习仪表盘（`/student/dashboard`）

| 区块 | 说明 |
|------|------|
| 统计卡片 | 近 7 日事件、错题总数、未掌握、掌握率 |
| **错题分析（ECharts）** | 掌握占比 / 来源 / 类别（三列）+ 趋势 / 知识点（双列）；随当前课程筛选 |
| 薄弱知识点 | 掌握度最低 Top5 |
| 近期活动 | 近 7 日最近 10 条事件 |
| 复习推荐 | 选课后展示 Top3 薄弱 KP 对应资料 |

侧栏入口：**学习仪表盘**。

### 2.2 错题本（`/student/wrong-book`）

| 能力 | 说明 |
|------|------|
| **类别 Tab** | 按 `syntax_error` / `semantic_error` 等自动归类筛选 |
| 列表 | 分页展示错题，支持按课程、掌握状态、类别筛选 |
| **详细分析** | 展开行查看 `issues`、`suggestions`、`review_tip` |
| 来源 | `代码讲解` / `AI 对话` |
| 标记掌握 | 切换 `mastered`，自动刷新知识点掌握度 |

可视化图表已移至 **学习仪表盘**「错题分析」区块；本页顶部提供跳转链接。

## 三、自动埋点（系统内部）

| 事件 | 触发模块 | 条件 | 错题本 |
|------|---------|------|--------|
| `code_submit` | M05 | 每次提交成功 | — |
| `code_analysis_error` | M05 | 分析 failed 或 syntax/semantic/runtime 为 error | 写入 `code_submission` 来源 |
| `chat_no_context` | M04 | 回答 `no_context=true` | 写入 `chat_message` 来源 |

M05 默认关联知识点 **「代码基础」**（需 `seed_knowledge_points.py` 种子）。

## 四、掌握度算法

服务：`backend/app/services/mastery.py`

```
对每个未掌握错题：
  weight = 1.0（code_submission）或 0.6（chat_message）
  penalty = weight × exp(-days / 30)
kp_score = max(0, 100 - sum(penalty) × 20)
```

标记已掌握后该条不再计入 penalty。单元测试见 `tests/test_mastery.py`。

## 五、推荐规则

`GET /api/v1/learning/recommendations?course_id=`：

1. 校验学生已选课（否则 `40301`）
2. 取掌握度最低 Top3 KP
3. 每 KP 推荐 1 条同课 `READY` 资料（按 id 倒序）
4. 无 KP 数据时按课程知识点 sort_order 兜底

**不使用 LLM 排序**。

## 六、API 一览

| 方法 | 路径 | 角色 | 说明 |
|------|------|------|------|
| POST | `/api/v1/learning/events` | student | 手动埋点（少用） |
| GET | `/api/v1/learning/dashboard` | student | 仪表盘（Redis 缓存 10min） |
| GET | `/api/v1/learning/wrong-book/stats` | student | 错题统计；`course_id?`, `days?`（默认 30） |
| GET | `/api/v1/learning/wrong-book` | student | 分页；`course_id?`, `mastered?`, `category?` |
| PUT | `/api/v1/learning/wrong-book/{id}/mastered` | student | Body: `{ mastered: bool }` |
| GET | `/api/v1/learning/recommendations` | student | `course_id` 必填 |
